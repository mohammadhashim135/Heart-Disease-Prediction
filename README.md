# ❤️ Heart Disease Prediction using Logistic Regression

## Overview

This project predicts the 10-year risk of Coronary Heart Disease (CHD) using clinical data from the Framingham Heart Study dataset.

A Logistic Regression model is trained to classify whether a patient is at risk (1) or not at risk (0).

The objective is to build a reliable and interpretable machine learning model for early heart disease detection.

---

## Features 🚀

- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature Scaling using StandardScaler  
- Logistic Regression with Class Balancing  
- Model Evaluation (Accuracy, Precision, Recall, F1-score)  
- Confusion Matrix Visualization  
- ROC Curve & AUC Score  
- Cross-Validation  

---

## Tech Stack 🛠

- **Language:** Python  
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn  
- **Environment:** Jupyter Notebook (.ipynb)

## Installation & Setup 🏗

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/mohammadhashim135/Heart-Disease-Prediction.git
cd Heart-Disease-Prediction
```

### 2️⃣ Install Required Libraries

```bash
pip install -r requirements.txt
```

### 3️⃣ Open Jupyter Notebook

```bash
jupyter notebook
```

### 4️⃣ Run the Notebook

Open and run:

```
heart_disease_prediction.ipynb
```
